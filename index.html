HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML provides content and basic framework including Text Elements, Image Elements, Header Elements, and List Elements.
  • In addition to elements that can be seen on the page, elements are also used to drivide the content on the page and to describe the content to browswers and other developers
  • Element types that are used to divide and describe a page include Meta-related elements, Content division elements, and Semantic elements

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • .card is a class selector
  • A border property assigns a border that is solid
  • A box-shadow CSS property adds shadow effects around an element's frame

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git init: initializes an existing directory as a Git repository
  • git clone: retrieves an entire repository from a hosted location via URL
  • git branch: lists your branch. a * will appear next to the currently active branch
  • git checkout: switches to another branch and checks it out into your working directory
  • git log: show all commits in the current branch's history
  • git push: Transmits local branch commits to the remote repository branch
  • git pull:fetch's and merge's any commits from the tracking remote branch

JavaScript

  • JavaScript is a programming language that allows users to interact with websites
  • Variables are containers that store values. You start by declaring a variable with the let keyword, followed by the name you give to the variable
  • var is the Variable keyword
  • Semicolons should be put at the end of a line of code to indicate where a statement ends
  • Control flow is the order in which a computer executes code in a script.
